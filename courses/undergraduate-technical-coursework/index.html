Undergraduate Technical Coursework

This page organizes my undergraduate technical coursework from the UESTC-University of Glasgow joint programme. It includes professional courses related to electrical engineering, electronics, embedded systems, computer systems, AI, signal processing, and hardware/software design. Political, humanities, law, business, and other non-technical courses are intentionally excluded.

The course files are kept locally; this page is a public course index and basic knowledge map. It does not publish private assignments, exams, answer sheets, or original course documents.

Basic Knowledge Map

Introductory Programming

  • Programming basics: variables, expressions, control flow, functions, arrays, strings, structures, and modular code organization.
  • Problem solving: decomposition, edge cases, debugging, test cases, and translating a small specification into executable logic.
  • Engineering habits: readable naming, incremental development, compiler/runtime errors, and basic versioned source-code thinking.

MS / Embedded Microcontroller Project

  • Embedded platform basics: microcontroller I/O, GPIO, timers, interrupts, PWM, SPI, and hardware-software interaction.
  • C/C++ on devices: memory constraints, peripheral configuration, polling versus interrupt-driven control, and state-machine design.
  • Project integration: connecting input, output, game logic, physical circuit behavior, and board-level debugging into one embedded application.

Digital Image Processing

  • Image representation: pixels, color spaces, grayscale images, histograms, spatial-domain and frequency-domain views.
  • Core operations: enhancement, filtering, convolution, image transforms, edge/feature extraction, segmentation, and object description.
  • Implementation tools: MATLAB/OpenCV workflows, GUI-based visualization, batch processing, and evaluation through visual and quantitative comparison.

CAD and Engineering Design Tools

  • Design representation: engineering drawings, dimensions, tolerances, part modeling, assemblies, and technical documentation.
  • Workflow basics: requirement-to-design translation, iterative revision, data table handling, and design-manual preparation.
  • Engineering value: making designs communicable, reproducible, and verifiable through structured CAD files and documentation.

Embedded Processor

  • Processor fundamentals: ARM architecture, registers, instruction execution, assembly/machine language, memory-mapped I/O, and interrupt handling.
  • Peripheral interfaces: GPIO, PWM, UART, SPI, I2C, ADC/DAC-style interfacing, polling, and hardware/software timing.
  • Embedded software design: device drivers, low-level debugging, firmware structure, and real-time interaction with external hardware.

Fundamentals of Analog Circuits

  • Circuit theory: Kirchhoff laws, nodal/mesh analysis, Thevenin/Norton equivalents, frequency response, and small-signal reasoning.
  • Analog devices: diodes, BJTs, MOSFETs, operational amplifiers, biasing, gain, impedance, and amplifier stages.
  • Lab and design: measurement, simulation, circuit validation, noise/sensitivity awareness, and translating a circuit idea into a working prototype.

Analog and Digital Design Laboratory

  • Digital logic: Boolean algebra, combinational circuits, sequential circuits, finite-state machines, counters, registers, and timing.
  • HDL and verification: Verilog modules, testbenches, simulation, synthesis awareness, and hardware debugging.
  • Mixed lab ability: combining analog measurement, digital control, and microprocessor-related logic into lab-scale hardware systems.

Electronic Devices

  • Semiconductor basics: bands, carriers, doping, PN junctions, drift/diffusion, recombination, and device I-V characteristics.
  • Device families: diodes, BJTs, MOSFETs, optoelectronic devices, and solid-state device behavior.
  • Engineering interpretation: using device physics to explain threshold, leakage, breakdown, capacitance, switching, and measurement results.

Electronic System Design

  • System-level electronics: requirements, block diagrams, power supply, sensing, signal conditioning, actuation, and interface design.
  • Circuit integration: analog front-end, digital control, microelectronic circuits, PCB-aware thinking, and verification planning.
  • Project workflow: specification, simulation, implementation, testing, documentation, and design review.

Machine Learning and Artificial Intelligence

  • ML foundations: supervised/unsupervised learning, features, labels, loss functions, optimization, train/validation/test splits, and metrics.
  • Model families: regression, classification, clustering, decision trees, neural networks, and basic probabilistic reasoning.
  • Practical workflow: Python environment setup, Jupyter notebooks, data preprocessing, model selection, overfitting control, and experiment reporting.

Signals and Systems

  • Signal models: continuous/discrete time, periodicity, impulse/step signals, linear time-invariant systems, convolution, and impulse response.
  • Transform methods: Fourier series, Fourier transform, Laplace/Z-transform basics, frequency response, and filtering interpretation.
  • System thinking: stability, causality, time/frequency-domain tradeoffs, and using mathematical tools to reason about physical signals.

Power Electronics

  • Power conversion basics: switching devices, rectifiers, buck/boost converters, DC-DC conversion, duty cycle, and efficiency.
  • Circuit behavior: inductor/capacitor energy storage, ripple, conduction modes, losses, thermal constraints, and control loops.
  • Design mindset: balancing power quality, efficiency, device stress, switching frequency, and practical implementation limits.

Dynamics and Control

  • Dynamic modeling: differential equations, transfer functions, state-space concepts, linearization, and system response.
  • Control basics: feedback, stability, transient/steady-state response, PID-style reasoning, root-locus/frequency-domain intuition, and disturbance rejection.
  • Lab interpretation: connecting measured system behavior to model parameters, controller tuning, and performance requirements.

Digital Signal Processing

  • Discrete signals: sampling, quantization, aliasing, discrete-time systems, convolution, and difference equations.
  • Frequency analysis: DFT/FFT, spectral leakage, windowing, digital filter design, FIR/IIR filters, and frequency response.
  • Implementation: MATLAB-based DSP pipelines, signal generation/recognition, filtering, transform-domain reasoning, and performance/accuracy tradeoffs.

Team Design Project Skills

  • Engineering teamwork: requirement analysis, task division, interface definition, risk tracking, and design documentation.
  • Embedded integration: STM32-style development, sensors, motors, communication modules, power, mechanical constraints, and debugging.
  • Delivery workflow: prototype iteration, testing, report writing, presentation, and turning a hardware idea into a complete demonstrator.

Electromagnetic Fields and Waves

  • Field fundamentals: electric/magnetic fields, Maxwell’s equations, boundary conditions, wave propagation, and energy flow.
  • Transmission concepts: plane waves, reflection, refraction, impedance, transmission lines, and antenna/waveguide intuition.
  • Engineering use: linking field theory to circuits, communication systems, microwave behavior, and electromagnetic compatibility.

Real-Time Computing Systems and Architecture

  • Real-time concepts: tasks, deadlines, latency, jitter, scheduling, priority, response-time analysis, and resource sharing.
  • Architecture basics: processor organization, memory hierarchy, interrupts, timers, and hardware features that affect timing.
  • System design: matching timing requirements to scheduling policy, implementation structure, and verification method.

Information Security

  • Security principles: confidentiality, integrity, availability, authentication, authorization, and threat modeling.
  • Cryptography basics: symmetric/asymmetric encryption, hashing, digital signatures, key exchange, and protocol-level reasoning.
  • System security: vulnerabilities, access control, network/security labs, secure design habits, and risk-aware engineering judgment.

VLSI Design

  • CMOS fundamentals: MOS transistors, logic gates, inverter characteristics, noise margins, delay, power, and scaling.
  • Digital IC design: combinational/sequential logic, timing constraints, layout concepts, clocking, and design-rule awareness.
  • Hardware flow: schematic-to-layout thinking, verification, synthesis/place-and-route awareness, and area-power-performance tradeoffs.

Computer Science Core Courses

  • Data structures and algorithms: arrays, lists, stacks, queues, trees, graphs, hashing, sorting, searching, and complexity analysis.
  • Computer organization and operating systems: CPU datapath, memory hierarchy, processes, threads, scheduling, synchronization, virtual memory, and file systems.
  • Computer networks and distributed computing: layered protocols, TCP/IP, routing, congestion, client-server models, parallelism, and distributed-system coordination.

Technical Thread

The undergraduate curriculum formed a broad electrical and computer engineering base. Circuit and device courses built the electronics foundation; embedded processor, TDPS, VLSI, RTCSA, and information security connected hardware to system design; signals, DSP, electromagnetic fields, power electronics, dynamics/control, image processing, and ML/AI built the modeling and algorithmic layer.

Together, these courses supported later work in efficient AI systems, GPU programming, hardware-software co-design, and edge-side intelligent systems.
