Postgraduate Technical Coursework

This page organizes my postgraduate technical coursework. It only includes courses and materials related to computer science, electronic and computer engineering, AI systems, security, architecture, and data systems. General education, ethics, humanities, and administrative materials are intentionally excluded.

The course files are kept locally; this public page records the technical scope and learning focus without publishing private notes, assignments, exams, or original course documents.

Course Catalog

Course Period Technical Focus Organized Materials
Algorithm Design and Analysis 2025-2026 Graph algorithms, divide and conquer, dynamic programming, greedy algorithms, backtracking, branch and bound, approximation algorithms, max-flow, linear programming, probabilistic algorithms, and complexity analysis. Lecture slides, textbook references, review notes, homework folders, and implementation code.
Security Chips 2025-2026 Trusted computing, secure processor architecture, TPM, HSM, RFID security, PUF, side-channel attacks, fault attacks, memory security, and Intel SGX. Lecture slides, secure processor readings, course report, and paper-reading materials.
Intelligent Computing System 2025-2026 Deep learning applications, programming frameworks, AI processors, accelerator architecture, intelligent programming languages, and LLM computing systems. Lecture slides, lab folders, exam review notes, and presentation/report materials.
Distributed Multimedia System and Technology 2025-2026 Distributed multimedia systems, media scheduling, video processing, video super-resolution, system design, and multimedia service workflows. Lecture slides, review notes, design reports, and system-case materials.
Big Data Analysis and Mining 2025-2026 Big data processing, data mining workflows, HBase, distributed data storage, and analytics-oriented system design. Lecture slides, textbook references, homework folders, and review materials.
Deep Learning 2025-2026 Neural network training, optimization, representation learning, deep learning systems, and lab-based model implementation. Lecture slides, lab materials, and review resources.
GPU Architecture and Programming 2025-2026 CUDA programming, GPU execution model, memory hierarchy, PTX, massively parallel processors, GPGPU architecture, and performance-oriented programming. CUDA references, GPU architecture readings, review notes, project materials, and presentation drafts.
Computer Systems Architecture 2025-2026 Quantitative computer architecture, ISA and microarchitecture, pipelining, memory hierarchy, parallelism, architecture evaluation, and system-level performance tradeoffs. Architecture textbooks, lecture slides, homework folders, and project materials.

Technical Thread

These courses connect around one broad direction: efficient AI and computing systems. Algorithm Design and Analysis provides the formal problem-solving base; Computer Systems Architecture and GPU Architecture and Programming provide the hardware and performance foundation; Intelligent Computing System, Deep Learning, and Big Data Analysis and Mining connect that foundation to AI workloads and data systems; Security Chips adds a hardware-security view of trustworthy computation.

The strongest overlap with my current research interests is in GPU systems, accelerator-aware AI systems, LLM computing infrastructure, and secure or reliable system design.
