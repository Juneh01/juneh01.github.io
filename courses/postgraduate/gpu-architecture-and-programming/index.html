GPU Architecture and Programming

学习阶段： 2025-2026
课程方向： GPU/SIMT 架构、CUDA、存储层次、Tensor Core 与推理优化

Course Overview

课程从 GPU 微架构和 SIMT 执行出发，覆盖 CUDA 编程、存储层次、并行模式、PTX、Tensor Core、性能分析，以及面向神经网络和大模型算子的系统优化。

Complete Syllabus

  1. 并行计算基础：吞吐导向设计、CPU/GPU 差异、Amdahl 和并行扩展。
  2. GPU 架构：SM、warp scheduler、执行管线、寄存器文件和线程束。
  3. CUDA 执行模型：host/device、kernel、grid、block、warp 和 thread。
  4. 线程组织与同步：block 内同步、原子操作、warp-level primitive 和协作组。
  5. 存储层次：register、shared、L1/L2、global、constant、texture 和 local memory。
  6. 访存优化：coalescing、bank conflict、对齐、缓存复用、tiling 和 latency hiding。
  7. 并行算法模式：reduction、scan、histogram、stencil、transpose 和矩阵乘。
  8. 占用率与资源约束：寄存器、shared memory、block size 和并发。
  9. Tensor Core 与 WMMA：混合精度矩阵乘、tile 映射和数据布局。
  10. PTX 与底层执行：指令级观察、inline PTX 和编译生成代码。
  11. CUDA Graph、流水线和异步拷贝：降低 launch 开销并重叠计算与传输。
  12. 性能模型与剖析：roofline、带宽/计算瓶颈、Nsight 指标和优化闭环。
  13. Triton/TileLang：高层 GPU kernel 编程与自动化映射。
  14. 大模型算子：Paged Attention、KV cache、vLLM 扩展和推理服务优化。

Labs and Projects

  • SCNN CUDA 推理框架优化：批处理、流水线、WMMA、CUDA Graph 和 PTX。
  • Qwen 部署与 GPU 编程问答系统。
  • Triton/TileLang 加速 vLLM Paged Attention 算子。
  • GPU 性能分析、内存复用和 kernel 级调优。

Knowledge Outcomes

  • 能够从 SIMT、存储层次和资源约束解释 GPU 性能。
  • 能够设计、实现和剖析 CUDA kernel。
  • 能够把模型推理瓶颈映射到 kernel、运行时和服务层优化。
