Machine Learning and AI - 混合优化与分类实验

项目概览

Machine Learning and AI 课程项目包含两条主要实践线索：一条是基于机器学习与遗传算法的混合优化，另一条是分类模型实验。项目材料中包含 MATLAB 代码、notebook、报告、训练数据和可视化结果。

这个项目的重点不是只训练一个模型，而是把机器学习模型和优化算法结合，用模型预测、搜索和评价函数共同求解工程设计问题。

Thermistor MixedIntegerGA

Thermistor 相关代码使用 MATLAB 实现，目标是通过混合整数遗传算法搜索热敏电阻电路中的元件组合，使输出曲线尽量匹配目标温度补偿曲线。

相关文件包括：

  • RunThermistorExample.m
  • hybridalgorithm.m
  • objectiveFunction.m
  • trainmodel.m
  • voltageCurve.m
  • tempCompCurve.m
  • StandardComponentValues.mat

整体流程可以拆成：

  1. 定义电路结构和标准元件值。
  2. 生成或读取训练数据。
  3. 训练预测模型或评价函数。
  4. 用遗传算法搜索离散元件组合。
  5. 绘制最优解曲线并分析误差。

这个问题很适合作为 ML + optimization 的练习，因为元件选择不是连续优化，存在离散变量、工程约束和目标曲线拟合。

Hybrid GA + Classifier

另一个方向是将遗传算法与分类器结合，用于特征选择或模型优化。项目材料中包含 SVM、Naive Bayes、Random Forest 等模型的实验 notebook。

实验关注的问题包括：

  • 不同分类器在同一数据集上的表现差异。
  • 是否使用遗传算法进行特征选择。
  • 标准化对分类结果的影响。
  • K-fold 或 train/test split 下的稳定性。

这些实验让我熟悉了机器学习项目中常见的模型比较方式，也认识到特征选择、数据预处理和评价指标往往和模型本身同样重要。

Microarray Classification

Microarray 相关材料用于高维小样本分类场景。微阵列数据通常特征维度高、样本量小，非常容易过拟合，因此适合用来练习特征选择和模型泛化分析。

在这类任务中，遗传算法可以作为特征子集搜索策略，分类器则负责评价当前特征组合的预测效果。

我的收获

这个项目让我把机器学习从“调用分类器”推进到“构造搜索空间和评价函数”。尤其是 Thermistor 优化部分，让我看到 ML 方法可以嵌入工程设计流程，而不是只用于图像或文本任务。

后续如果继续完善，我会把 MATLAB 结果图和关键参数表整理出来，并补充不同分类器的准确率、稳定性和特征选择对比。
