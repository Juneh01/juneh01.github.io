2026.03 - 2026.06 · Featured · AI systems / VLM inference

AICAS 2026 PPU Track: Qwen3-VL Inference Optimization

Optimized fixed-weight Qwen3-VL-2B-Instruct inference on Alibaba PPU-ZW810E through profiling-driven visual preprocessing, prompt KV reuse, load-time W8 layouts, persistent decode kernels, synchronization reduction, shared-memory staging, and call-graph trimming.

2026.01 · Featured · LLM application / GPU knowledge system

GPU-Tutor: Intelligent Q&A System for GPU Programming

Built a domain-specific Q&A system for GPU architecture and CUDA programming using Qwen3-0.6B, SFT + DPO, multi-index RAG routing, FAISS retrieval, vLLM serving, continuous batching, prefix caching, and embedding caching.

2025.12 · Featured · CUDA / inference acceleration

SCNN CUDA Inference Optimization

Designed a CUDA inference optimization path for spiking convolutional neural networks, combining batching, pipelining, Ping-Pong buffering, WMMA Tensor Core kernels, CUDA Graph, PTX inline instructions, and timestep-aware retraining.

2023.05 - 2025.06 · Featured · 6DoF pose / accelerator hardware

PRADO: High-Efficiency Pose Refinement Accelerator

Worked on hardware-oriented acceleration for high-precision RGB-D 6DoF pose refinement, including fast Sigmoid and SoftMax hardware modules, post-layout simulation verification, and compression-aware accelerator exploration.

2024 - 2025 · Featured · 6DoF pose / lightweight vision

Final Year Project: Lightweight 6DoF Pose Estimation

Explored HiPose-DG by adding dynamic spatial and channel gating into RGB-D 6DoF pose estimation, covering BOP dataset preparation, Docker-based training, LMO evaluation, and accuracy-efficiency analysis.

2025.03 · Featured · edge LLM / hardware-software co-design

IEEE AICAS 2025 Grand Challenge: LLM System Optimization

Optimized Qwen2.5-0.5B edge inference with INT4 quantization and a T-MAC + llama.cpp pipeline, focusing on memory reduction, decoding throughput, and deployment-oriented low-bit inference behavior.

2024.01 · Featured · mathematical modeling · Awarded 2024.05

MCM/ICM 2024: Deep-Sea Rescue Modeling

Built a rescue decision framework for submersible trajectory prediction, equipment allocation, and dynamic search strategy using ARIMA, ridge regression, genetic algorithms, Poisson modeling, Bayesian updates, and grid-based search.
