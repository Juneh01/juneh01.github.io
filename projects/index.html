Selected Projects

PRADO: High-Efficiency Pose Estimation Accelerator

UESTC 6DoF Research Group, 2023.05 - 2025.06

Worked on hardware-oriented acceleration for high-precision 6DoF pose estimation in VR/AR and robotic edge scenarios. Implemented fast activation-function hardware modules including Sigmoid and SoftMax, and participated in post-layout simulation verification and compression-aware experiments.

IEEE AICAS 2025 Grand Challenge: LLM System Optimization

Alibaba T-Head Yitian 710, 2025.03

Optimized Qwen2.5-0.5B on edge hardware using INT4 quantization and a T-MAC + llama.cpp inference pipeline. The recorded result reduced memory usage by 75% and improved decoding throughput by about 7x.

SCNN CUDA Inference Optimization

UCAS GPU Architecture and Programming Course Competition, 2025.12

Designed and optimized a CUDA inference framework for spiking convolutional neural networks. Applied batching, pipelining, Ping-Pong buffering, WMMA Tensor Core, CUDA Graph, PTX inline instructions, and timestep-aware retraining. The inference time was reduced from 5.64s to 15.4ms while maintaining 89.89% accuracy.

GPU-Tutor: Intelligent Q&A System for GPU Programming

UCAS GPU Architecture and Programming Course Competition, 2026.01

Built a domain-specific question answering system for GPU architecture and programming. The system combined Qwen3-0.6B fine-tuning, SFT + DPO, multi-index RAG routing, FAISS retrieval, vLLM serving, continuous batching, prefix caching, and embedding caching.

Final Year Project: Lightweight 6DoF Pose Estimation

UESTC / Glasgow College, 2024 - 2025

Explored a lightweight HiPose-DG framework by adding dynamic spatial and channel gating into RGB-D 6DoF pose estimation. The project involved BOP dataset preparation, Docker-based training, LMO evaluation, and accuracy-efficiency analysis.

MCM/ICM 2024: Deep-Sea Rescue Modeling

Outstanding Winner & INFORMS Award, 2024

Built a multi-model rescue decision framework for submersible trajectory prediction, equipment allocation, and dynamic search strategy using ARIMA, ridge regression, genetic algorithm, Poisson distribution, and Bayesian update.
