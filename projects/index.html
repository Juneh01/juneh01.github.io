Project Timeline

GPU-Tutor: Intelligent Q&A System for GPU Programming

2026.01 · UCAS GPU Architecture and Programming Course Competition

Built a domain-specific Q&A system for GPU architecture and CUDA programming with Qwen3-0.6B fine-tuning, SFT + DPO alignment, multi-index RAG routing, FAISS retrieval, vLLM serving, continuous batching, prefix caching, and embedding caching. The system ranked 4th in the Paratera Tech Track with 0.3733 ROUGE-L and 33,000+ chars/s inference speed.

SCNN CUDA Inference Optimization

2025.12 · UCAS GPU Architecture and Programming Course Competition

Designed and optimized a CUDA inference framework for spiking convolutional neural networks. The work combined batching, pipelining, Ping-Pong buffering, WMMA Tensor Core kernels, CUDA Graph, PTX inline instructions, and timestep-aware retraining, reducing inference time from 5.64s to 15.4ms while maintaining 89.89% accuracy.

PRADO: High-Efficiency Pose Refinement Accelerator

2023.05 - 2025.06 · UESTC 6DoF Research Group

Worked on hardware-oriented acceleration for high-precision 6DoF pose estimation in VR/AR and robotic edge scenarios. Implemented fast activation-function hardware modules including Sigmoid and SoftMax, and participated in post-layout simulation verification and compression-aware experiments.

Final Year Project: Lightweight 6DoF Pose Estimation

2024 - 2025 · UESTC / Glasgow College

Explored a lightweight HiPose-DG framework by adding dynamic spatial and channel gating into RGB-D 6DoF pose estimation. The project involved BOP dataset preparation, Docker-based training, LMO evaluation, and accuracy-efficiency analysis.
